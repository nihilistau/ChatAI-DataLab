# Kitchen Onboarding Guide

Welcome to the Kitchen Playground! This guide will help you get started:

## What is Kitchen?
- Kitchen is a modular notebook system for building, chaining, and sharing Recipes (notebook pages) and Cookbooks (collections).

## Getting Started
1. Explore example notebooks in `kitchen/notebooks/`.
2. Open `welcome_cookbook.ipynb` for a guided tour.
3. Try running cells and chaining outputs.
4. Create your own Recipe by adding a new notebook.

## Extending Kitchen
- Add new Recipes: Create a notebook in `kitchen/notebooks/`.
- Build Cookbooks: Group related Recipes together.
- Use extension templates in `kitchen/widgets/` and `kitchen/scripts/`.
